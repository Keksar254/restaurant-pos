# 🍽️ Restaurant POS System

A modern, offline-first Point of Sale system for restaurants with AI-powered upselling, QR code receipts, and real-time inventory management.

## ✨ Features

- 📴 **Offline-First** - Works without internet, syncs when back online
- 🤖 **AI Upselling** - Smart recommendations based on cart items
- 📱 **QR Receipts** - Digital receipts with QR codes
- 💳 **Multiple Payment Methods** - Cash, Credit Card, M-Pesa
- 📦 **Real-time Inventory** - Stock updates instantly
- 🎨 **Beautiful UI** - Modern, responsive design

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Keksar254/restaurant-pos.git
cd restaurant-pos

# Install dependencies
npm install

# Start the server
node server.js
